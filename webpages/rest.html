rest

This example demonstrates how to make all four kinds of REST calls - GET, POST, PUT, and DELETE.

An example of a REST GET call: 

GET apis/rest/ant/bat?p3=cat&p4=dog

An example of a REST POST call: 

POST apis/rest/ant/bat?p3=cat&p4=dog

An example of a REST PUT call: 

PUT apis/rest/ant/bat?p3=cat&p4=dog

An example of a REST DELETE call: 

PUT apis/rest/ant/bat?p3=cat&p4=dog
