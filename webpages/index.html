Webpages

Lots of webpages that demonstrate how to call to the backend to get data. Lots of examples of layouts and how to display data.

Name Description
rest Demonstrates how to make calls to all four kinds of REST methods - GET, POST, PUT, and DELETE.
news Shows top stories from an rss new feed.
geoplace Shows information about a location. E.g. "Palo Alto" or "SFO".
maps Shows the location of a place (e.g. "SFO" or "Palo Alto") on a Google map.
flickr Shows images whose tags match a search term. Note: requires that a flickr api key be set in the api_keys.dat file.

REST APIs

Some simple examples of how to implement a REST call to support mobile apps.

Name Description
rest Demonstrates how to handle all four types of REST calls - GET, POST, PUT, and DELETE.
geoplace Shows geo information about a location. E.g. "Palo Alto" or "SFO". Demonstrates how to make a YQL call in the backend.
flickr Searches flickr for photos. Demonstrates how to use a flickr API key that's stored behind the REST API.
